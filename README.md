# right-pad

A very simple npm package that adds a space to the right of a string.

[한국어 문서 (Korean)](#한국어-문서)

---

## Installation

```bash
npm install right-pad
```

## Usage

```typescript
import rightPad from 'right-pad';

const result = rightPad('hello');
console.log(`'${result}'`); // 'hello '
```

## Features

- Written in TypeScript
- Zero dependencies
- Extremely lightweight
