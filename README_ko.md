# right-pad `v1.0` 🔥

> **가장 빠르고, 가장 강력하다. 오직 속도뿐.**

고성능 문자열 패딩의 글로벌 표준. 절대적인 속도와 제로 오버헤드를 위해 설계되었습니다.

## ⚡️ 초고속 시작 (NPM 전용)

```bash
npm i right-pad
```

## 🛠 극한의 사용법 (단 한 단어)

```typescript
import r from 'right-pad';

r('최적화'); // '최적화 '
```

## 🏎 성능 벤치마크: GOD MODE
- **시간 복잡도**: **O(1)** (이론적 한계치)
- **의존성**: **0.0%** (순수 그 자체)
- **메모리**: **나노 스케일** 점유율
- **엔진**: **V8 JIT / Bun / Deno** 최적화 완료
